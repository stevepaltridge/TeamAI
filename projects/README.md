# Projects

This folder contains project templates and starter scaffolds for common work across the team.

---

## Using a Template

1. Copy the template folder to your working location.
2. Rename it to match your project.
3. Follow the `README.md` inside the template folder for setup steps.

---

## Adding a Template

1. Create a new folder under `projects/` named after the template (e.g. `projects/python-script-starter/`).
2. Include a `README.md` inside describing:
   - What the template is for.
   - How to set it up.
   - Any dependencies required.
3. Open a Pull Request so the team can review and use the template.
