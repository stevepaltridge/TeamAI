Automation & Intelligence

Automation Scripts &
External Intelligence

Four Core Scripts

ice-api.js REST

201 lines — REST API Proxy

OAuth2 password-grant authentication with automatic token management. Commands: token, get, folders, field, settings, rules, schema.

Supports parallel requests with per-session caching. Primary entry point for all REST-based data retrieval.

ice-fetch.js WEB

Playwright Auth-Walled Fetcher

Automated browser login through ICE identity provider (IDP). Handles OAuth2 redirect flow, extracts page content from authenticated Encompass web sessions.

Used for scraping admin portal pages not available via REST API.

build-knowledge-base-v2.js SDK

1,115 lines — KB Generator

Master generator that loads ~30 data sources and produces 24+ structured markdown KB sections. Cross-references data across users, orgs, personas, rules, fields, and templates.

Outputs to knowledge/encompass-settings/ with deterministic naming.

extract-live.js REST

262 lines — Live Data Extraction

Authenticated REST API extractor using OAuth2 + fetchAndSave pattern. Pulls real-time data from production instance and saves structured JSON to data/be11146040/.

Supports incremental updates and differential extraction.
