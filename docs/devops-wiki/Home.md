## What Is TeamAI?

TeamAI is our AI-powered knowledge system. It connects VS Code + GitHub Copilot to:

- **12,700+ MasteringEncompass articles** (indexed, searchable)
- **1,018 EllieForum community posts**
- **All company SharePoint and OneDrive documents**
- **Live Encompass API** (loan folders, fields, schemas)
- **ICE Developer Connect** (auth-walled docs)

You ask a question in Copilot Chat. The AI searches all of these sources and gives you an answer with citations.
